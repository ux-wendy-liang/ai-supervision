# Coach/Therapist 筛选条件竞品分析

## 研究目的
分析市场上主要的心理健康、教练辅导平台的筛选功能，找出用户在寻找 Coach 时最重要的筛选条件。

---

## 一、核心发现：最重要的筛选维度

根据用户研究数据，选择 Coach/Therapist 时最重要的因素排序：

| 排名 | 筛选因素 | 用户认可度 |
|------|---------|-----------|
| 1 | 个人连接感 (Personal Connection) | 67.7% |
| 2 | 专业领域 (Areas of Expertise) | 62.7% |
| 3 | 帮助意愿 (Commitment to helping) | 55.0% |
| 4 | 咨询师类型 (Type of therapist) | 52.7% |
| 5 | 亲和力 (Warmth) | 50.3% |
| 6 | 是否接受保险 (Insurance) | 46.2% |
| 7 | 费用 (Cost) | 44.2% |
| 8 | 循证治疗专长 (Evidence-based expertise) | 35.9% |

> 数据来源: [PMC Consumer Expectations Survey](https://pmc.ncbi.nlm.nih.gov/articles/PMC10288349/)

## 二、各平台筛选条件对比

### 1. Psychology Today（心理咨询目录龙头）

**平台特点**: 最大的心理健康服务商目录，Google搜索结果排名第一达 96.2%

**筛选维度**:
- **地理位置**: 州、城市、邮编
- **问题类型**: 焦虑、抑郁、创伤、关系等（可展开更多选项）
- **咨询师性别**: 男/女
- **服务形式**: 线上 / 线下
- **保险类型**: 支持的保险计划列表
- **治疗方法**: CBT、精神分析等不同流派
- **价格范围**: 包含滑动价格选项
- **可预约状态**: 是否接受新客户

**亮点功能**:
- 视频介绍（帮助评估是否合适）
- "Popular Filters" 快速筛选

> 来源: [Psychology Today Directory](https://www.psychologytoday.com/us/therapists)

---

### 2. ADPList（设计/职业导师平台）

**平台特点**: 免费职业导师平台，覆盖 UX、产品、AI、开发等领域

**筛选维度**:
- **搜索框**: 姓名、职位、公司
- **专业领域**: 按技能/学科浏览
- **经验级别**: Lead、Manager 等
- **语言**: 导师使用的语言
- **时区**: 方便安排会议
- **可预约状态**: 显示日历空档

**亮点功能**:
- 智能匹配：根据问卷回答匹配导师
- 导师可以为老用户预留专属时间

> 来源: [ADPList Help Center](https://intercom.help/adplist-community/en/articles/8484298-how-to-find-and-book-a-mentor)

---

### 3. BetterHelp（在线心理咨询）

**平台特点**: 在线心理治疗平台，通过算法匹配

**筛选/匹配维度**:
- **地理位置**: 咨询师需要在用户所在州持有执照
- **可预约时间**: 根据偏好时段
- **专业领域**: 焦虑、抑郁等
- **用户偏好**:
  - 性别偏好
  - 年龄范围
  - 宗教信仰
  - 文化背景
- **治疗目标**: 从问题清单中选择

**亮点功能**:
- 免费无限次更换咨询师
- 可选择让系统匹配或自己浏览选择

> 来源: [BetterHelp FAQ](https://www.betterhelp.com/faq/)

---

### 4. Talkspace（在线心理咨询）

**平台特点**: 48小时内完成算法匹配

**筛选维度**:
- **州/地区**: 执照要求
- **专业类型**: 服务类型
- **保险**: 保险计划兼容性
- **性别偏好**: 唯一可主动选择的偏好

**局限性**:
- 不能选择咨询师的宗教信仰、性取向
- 不能选择特定认证或经验年限
- 不能查看 Live Session 可用性

> 来源: [Talkspace Help Center](https://help.talkspace.com/hc/en-us/articles/24477925465371-Match-with-a-Provider)

---

### 5. Headspace（心理健康教练）

**平台特点**: 结合冥想 App 的心理健康服务

**筛选/匹配维度**:
- **专业领域**: 教练的专长
- **服务形式**: 文字/视频等
- **种族**: 可选择同种族教练
- **性别**: 性别偏好
- **可预约状态**: 时间空档

**教练资质要求**:
- 国家认证 (NBHWC) 或硕士学位
- 2年以上经验
- 200+ 小时督导培训

**亮点功能**:
- 文化敏感性护理
- 强调身份认同和社会公正

> 来源: [Headspace Coaching](https://www.headspace.com/coaching)

---

### 6. TherapyDen（包容性咨询目录）

**平台特点**: 专注社会正义和包容性的免费目录

**筛选维度**: 140+ 筛选条件！

**特色筛选**:
- **身份认同类**:
  - 跨性别/非二元性别
  - LGBTQ+ 友好
  - BIPOC（黑人、原住民、有色人种）
  - 开放式关系/多元关系
- **费用**: 价格范围
- **咨询师身份**: 咨询师自身的性别、种族、LGBTQ身份

**亮点功能**:
- 专注服务传统上较难找到合适咨询师的群体
- 完全免费（对咨询师也免费）

> 来源: [TherapyDen](https://www.therapyden.com/)

---

### 7. AI 教练平台（CoachHub、BetterUp 等）

**平台特点**: 企业级 AI + 真人教练混合模式

**匹配维度**:
- **目标**: 发展目标
- **行业经验**: 教练的行业背景
- **文化因素**: 文化契合度
- **性格**: 性格匹配
- **发展需求**: 具体能力提升需求

**AI 匹配特点**:
- 分析沟通风格、教练方法论
- 基于过往成功案例模式匹配
- 语言偏好分析

> 来源: [AI Coaching Platforms 2025](https://honehq.com/resources/blog/10-best-ai-coaching-platforms-employee-development-2025/)

---

### 8. Calm Health（心理健康应用）

**平台特点**: 基于筛查问卷的个性化推荐

**匹配方式**:
- **心理健康筛查**: GAD-7（焦虑）、PHQ-9（抑郁）
- **自我报告目标**: 用户设定的目标
- **健康计划整合**: 与用户的保险福利对接

**亮点功能**:
- 个性化行动计划
- 根据筛查结果推荐不同级别的支持

> 来源: [Calm Health FAQ](https://support.calm.com/hc/en-us/articles/27054399791643-Calm-Health-FAQ)

## 三、筛选条件分类汇总

### A. 基础筛选（几乎所有平台都有）

| 类别 | 具体选项 |
|-----|---------|
| **专业领域** | 焦虑、抑郁、职业发展、人际关系、创伤等 |
| **服务形式** | 线上视频 / 线下面谈 / 文字聊天 |
| **地理位置** | 国家、州、城市、邮编 |
| **价格/费用** | 价格范围、滑动定价、免费咨询 |
| **可预约状态** | 是否接受新客户、日历空档 |

### B. 个人偏好（中等普及度）

| 类别 | 具体选项 |
|-----|---------|
| **性别** | 男 / 女 / 跨性别 / 非二元 |
| **语言** | 服务语言 |
| **年龄** | 咨询师年龄范围 |
| **种族/文化背景** | BIPOC、亚裔等 |
| **宗教信仰** | 基督教、佛教等 |

### C. 专业资质（高端/专业平台）

| 类别 | 具体选项 |
|-----|---------|
| **认证类型** | PhD、LCSW、LMFT、ICF 认证等 |
| **治疗方法** | CBT、DBT、精神分析、ACT 等 |
| **经验年限** | 执业年数 |
| **督导经验** | 培训小时数 |

### D. 身份认同（包容性平台特有）

| 类别 | 具体选项 |
|-----|---------|
| **LGBTQ+ 友好** | 是否专门服务 LGBTQ+ |
| **咨询师身份** | 咨询师自身的性别认同、性取向 |
| **特殊群体** | ADHD、神经多样性、开放式关系等 |

## 四、设计建议

### 1. 必须有的核心筛选（MVP）

```
优先级 1（必须）:
├── 专业领域/问题类型
├── 服务形式（线上/线下）
├── 价格范围
└── 可预约状态

优先级 2（重要）:
├── 语言
├── 性别偏好
└── 时区/地区
```

### 2. 差异化机会

根据竞品分析，可以考虑的差异化方向：

1. **更智能的匹配**:
   - 问卷式引导 + AI 推荐（类似 BetterUp）
   - 不只是筛选，而是"帮你找到最合适的"

2. **身份认同筛选**:
   - TherapyDen 的 140+ 筛选证明有需求
   - 特别是 LGBTQ+、文化背景等

3. **视频介绍**:
   - Psychology Today 的亮点功能
   - 帮助建立"个人连接感"（最重要的因素）

4. **透明度**:
   - 显示回复时间
   - 显示会话成功率/评价

### 3. 筛选 UI 模式

| 模式 | 适用场景 | 代表平台 |
|-----|---------|---------|
| **搜索 + 筛选** | 用户知道想要什么 | Psychology Today |
| **问卷引导** | 用户不确定需要什么 | BetterHelp |
| **AI 匹配** | 企业级、个性化 | CoachHub |
| **混合模式** | 兼顾不同用户 | ADPList |

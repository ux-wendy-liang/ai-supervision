# Coach 生态平台产品概念

> ADPList for Coach + AI Supervision

---

## 一、产品愿景

打造一个 **Coach 生态平台**，解决两端的问题：

| 用户 | 痛点 | 解决方案 |
|------|------|---------|
| **客户** | 找不到合适的 Coach | 目录 + 智能匹配 + 预约 |
| **Coach** | 缺乏持续成长支持 | AI Supervision 帮助成长 |

**一句话描述**：帮客户找到好 Coach，帮 Coach 变得更好。

---

## 二、为什么这个组合有意义？

### 2.1 形成飞轮效应

```
更多 Coach 入驻
      ↓
更多客户选择
      ↓
Coach 需要 AI Supervision 提升技能
      ↓
Coach 变得更好
      ↓
客户满意度提高
      ↓
```

|------|--------|--------|------------------|
| Noomii | ✅ | ❌ | ❌ |
| BetterUp | ✅ | ✅ | ❌ |
| ADPList | ✅ | ✅ | ❌ |
| **我们** | ✅ | ✅ | ✅ |

AI Supervision 是**独特价值**，其他平台没有。

## 三、目标用户

### 3.1 Coach（供给侧）

| 类型 | 特点 | 需求 |
|------|------|------|
| 新手 Coach | 刚拿认证，0-2年经验 | 获客 + 技能提升 |
| 成长期 Coach | 2-5年经验，想扩大影响力 | 获客 + 持续精进 |
| 资深 Coach | 5年+，已有稳定客源 | 品牌曝光 + 高端督导 |

### 3.2 客户（需求侧）

| 类型 | 场景 |
|------|------|
| 职场人士 | 职业发展、领导力、转型 |
| 创业者 | 业务决策、压力管理 |
| 个人成长 | 生活平衡、目标设定、习惯养成 |

## 四、核心功能

### 4.1 目录 + 预约（面向客户）

**参考 ADPList 的体验：**

| 功能 | 说明 |
|------|------|
| Coach Profile | 头像、简介、专长、认证、评价 |
| 筛选搜索 | 按领域、语言、价格、可用时间筛选 |
| 智能匹配 | 根据客户需求推荐合适 Coach |
| 日历预约 | 查看可用时间，直接预约 |
| 视频会议 | 内置或集成 Zoom/Google Meet |
| 评价系统 | Session 后客户可评价 |

### 4.2 AI Supervision（面向 Coach）

**核心功能：**

| 功能 | 说明 |
|------|------|
| Session 复盘 | 上传对话记录，获得 AI 反馈 |
| 模拟练习 | AI 扮演客户，练习教练对话 |
| 技能评估 | 分析教练能力，给出发展建议 |
| 困难案例咨询 | 讨论棘手客户情况 |
| 学习资源 | 根据评估推荐学习内容 |

### 4.3 Coach 管理后台

| 功能 | 说明 |
|------|------|
| 预约管理 | 查看、确认、调整预约 |
| 客户管理 | 记录客户信息和 session 笔记 |
| 收入统计 | 查看收入、提现 |
| 成长记录 | AI Supervision 的学习历程 |

## 五、商业模式

### 5.1 收入来源

| 来源 | 模式 | 说明 |
|------|------|------|
| **Coach 订阅** | $29-99/月 | 基础版免费，高级功能付费 |
| **平台抽成** | 10-20% | 通过平台完成的交易 |
| **AI Supervision** | $19-49/月 | 可单独订阅或捆绑 |
| **认证/培训** | 按项目 | 未来可拓展 |

### 5.2 定价策略（参考）

**Coach 套餐：**

| 套餐 | 价格 | 包含 |
|------|------|------|
| Free | $0 | 基础 Profile，3 个预约/月 |
| Pro | $29/月 | 无限预约，分析面板，15% 抽成 |
| Premium | $79/月 | Pro + AI Supervision，10% 抽成 |

**客户侧：**
- 浏览、搜索免费
- 预约按 Coach 定价（$50-300/session 不等）

## 七、MVP 建议

### 第一版（验证核心假设）

**只做最核心的功能：**

1. **Coach Profile 页面** - 展示 Coach 信息
2. **简单搜索/筛选** - 找到合适 Coach
3. **预约系统** - 选时间、预约 session
4. **AI Supervision（基础版）** - 1-2 个核心功能

**验证的假设：**
- Coach 愿意入驻吗？
- 客户会通过平台找 Coach 吗？
- Coach 会用 AI Supervision 吗？

### 暂不做的功能

- 支付系统（先让 Coach 自己收款）
- 视频会议（先用 Zoom）
- 复杂的评价系统
- 移动 App

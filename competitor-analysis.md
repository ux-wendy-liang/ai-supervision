## 七、核心 Takeaways

### 学到的成功模式

1. **SimCare 模式** — 模拟练习 + 即时反馈，$79/年简单定价，YC 验证
2. **RaeNotes 模式** — 专注单一痛点做到极致，按使用量计费
3. **Poised 模式** — 实时反馈是杀手功能，趋势追踪增加粘性
4. **Rocky 模式** — ICF 背书很重要

### 关键成功因素

1. **专注单一用户群** — 不要什么都做
2. **ICF 合规/背书** — 增加可信度
3. **简单定价** — 降低决策门槛
4. **B2B2C 渠道** — 通过培训机构触达终端用户
5. **即时反馈** — 核心差异化

### 要避免的坑

1. **UI 太复杂** — Rocky 用户反馈的问题
2. **「被监控」感** — Eleos 早期遇到的阻力
3. **定位模糊** — 替代 vs 补充 vs 练习工具要清晰

## 八、建议产品方向

### 首选：SimCare for Coaches

**概念：** AI 模拟「被教练者」，让 Coach 练习教练技巧

**功能：**
- 多种被教练者人设（职业挑战、人际问题、转型等）
- 基于 ICF Core Competencies 实时反馈
- Session 后详细分析报告
- 进度追踪和技能提升可视化

**定价：** $79-99/年 或 $10-15/月

**目标用户：**
- Coach 培训学员（最佳）
- 准备 ICF 认证的 Coach
- 想持续提升的执业 Coach

**为什么这个方向：**
1. SimCare 已验证模式有效（$4M 收入，YC 投资）
2. Coach 领域完全空白
3. 定价有参考（$79/年已被接受）

### 备选：Poised for Coaches

**概念：** 实时会议反馈工具，专为 Coach session 设计

**功能：**
- Session 中实时提示（提问质量、倾听比例、沉默运用等）
- 基于 ICF 能力模型评估
- 会后分析和趋势追踪

**定价：** $15-25/月

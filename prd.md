## 产品愿景

**一句话**：帮客户找到好 Coach，帮 Coach 变得更好。

**产品定位**：ADPList for Coach + AI Supervision = Coach 生态平台

```
┌─────────────────────────────────────────────────────────────┐
│                     Coach 生态平台                           │
├─────────────────────────────┬───────────────────────────────┤
│                             │                               │
│     客户端                   │         Coach 端              │
│     ────                    │         ──────                │
│     • 搜索/筛选 Coach        │     • Profile 展示            │
│     • 查看 Profile          │     • 预约管理                 │
│     • 预约 Session          │     • 客户管理                 │
│     • 评价反馈              │     • AI Supervision 成长      │
│                             │                               │
└─────────────────────────────┴───────────────────────────────┘
                              ↓
              形成飞轮：Coach 越好 → 客户越满意 → 更多 Coach 入驻
```

## 市场价值验证

### 这个生意有市场吗？

**结论：有，而且是双重机会。**

#### 教练行业市场数据（2025）

| 指标 | 数据 | 来源 |
|------|------|------|
| **全球教练市场规模** | $73 亿美元 | ICF 2025 全球研究 |
| **美国市场** | $160 亿美元 | ResearchAndMarkets |
| **全球Coach数量** | 167,000+ | ICF 推算 |
| **美国Coach数量** | 232,000+ | 行业报告 |
| **年增长率** | 17% CAGR | 2019-2023 数据 |

#### 两个市场机会

| 方向 | 竞争情况 | 机会 |
|------|---------|------|
| **AI Coach Supervision** | 几乎没有竞品 | 蓝海 |
| **Coach 目录平台** | 有 Noomii 等，但体验差 | 红海中有差异化机会 |

#### Coach 最大痛点：获客难

> 📄 详见 [coach-acquisition-pain.md](./research/coach-acquisition-pain.md)

**这是 Coach 的 #1 挑战：**

| 数据 | 说明 |
|------|------|
| 获客成本 (CAC) | **$200-500** / 客户 |
| 高端教练 CAC | **$1,500-5,000** |
| 销售周期 | **6-12 次接触** 才能成交 |
| 需要客户量 | **25-35 人** 才能全职做 |

**为什么获客这么难？**

| 原因 | 说明 |
|------|------|
| 不会营销 | Coach 培训只教教练技术，不教获客 |
| 市场饱和 | 美国 23 万+ Coach，同质化严重 |
| 自我营销尴尬 | Coach 就是产品，不好意思自夸 |
| 销售周期长 | 高价 + 信任依赖 = 决策慢 |

**"饥荒-盛宴"循环：**
```
这个月 5 个客户 → 下个月突然没有 → 完全靠转介绍 → 不可控、焦虑
```

#### 为什么组合成生态？

| 单独做 | 问题 | 组合做 |
|--------|------|--------|
| 只做目录 | 和 Noomii 竞争，没有差异化 | AI Supervision 是独特价值 |
| 只做 AI Supervision | 获客难，Coach 要自己找 | 目录带来自然流量 |

**目录如何解决获客难：**

| 传统方式 | 我们的平台 |
|---------|-----------|
| Coach 花时间做内容营销 | 只需填好 Profile |
| Coach 花钱打广告 | 平台统一获取流量 |
| 被动等转介绍 | 主动被客户发现 |
| 获客成本 $200-500 | 订阅费 $29-79/月 |

**为什么 Coach 愿意付费？**

> **$79/月 订阅费 < $200-500 获客成本**
>
> 如果平台能稳定带来客户，这是划算的。

#### 竞品对比

| 竞品 | 有目录 | 有预约 | 有 AI Supervision |
|------|--------|--------|------------------|
| Noomii | ✅ | ❌ | ❌ |
| BetterUp | ✅ | ✅ | ❌ |
| ADPList | ✅ | ✅ | ❌ |
| **我们** | ✅ | ✅ | ✅ ← 独特 |

#### 咨询师 vs Coach 的 AI 督导工具对比

**咨询师领域 - 已有成熟产品：**
- [Eleos Health](https://eleos.health/) - 临床督导，分析循证疗法
- [Lyssn](https://www.lyssn.io/) - 评估共情能力，即时反馈
- [Mentalyc](https://www.mentalyc.com/) - 督导笔记，洞察分析

**Coach 领域 - 空白！**
- CoachHub AIMY → 帮被教练的人，不是帮 Coach
- Rocky.ai → AI 替代 Coach，不是帮 Coach 成长
- Coachvox → 让 Coach 创建 AI 分身服务客户

**结论**：专门帮 **Coach 本身做督导/成长** 的 AI 产品几乎没有！

**CoachHub**（暂定，待确认是否有商标冲突）

## 二、目标用户画像（Personas）

### 用户 A：考虑转型做 Coach 的人 - 小陈

**基本信息：**
| 项目 | 内容 |
|------|------|
| 年龄 | 30-45 岁 |
| 背景 | HR / 培训师 / 管理者 / 心理咨询相关 |
| 阶段 | 听说过教练，感兴趣，但不确定自己适不适合 |
| 收入 | 稳定工作，考虑职业转型或发展副业 |

**特征：**
- 对"帮助他人成长"有热情
- 听朋友或同事提过教练，觉得自己可能适合
- 但不确定自己是否具备做 Coach 的特质
- 培训课程要几万块，不敢轻易报名

**核心问题：**
> "我适合做 Coach 吗？我有没有这方面的天赋？"

**需求：**
- 低成本/免费试一试教练对话是什么感觉
- 了解 Coach 需要什么能力
- 得到一个初步评估：我适不适合

**痛点：**
| 痛点 | 现状 | 我们的解决方案 |
|------|------|---------------|
| 不知道自己适不适合 | 只能听别人说，没法亲自体验 | 免费试用 AI 模拟对话 |
| 培训太贵不敢轻易报名 | 几万块的培训费，万一不适合呢 | 先体验，再决定 |
| 不了解 Coach 需要什么能力 | 网上信息零散 | 体验后给出"适合度评估" |

**适合的功能：** 功能 1（免费体验版）

**商业价值：** 这是获客漏斗的入口
```
免费体验 → 发现自己适合 → 去报培训班 → 成为付费用户
```

### 用户 B：Coach 培训学员 - 小李（核心用户）

**基本信息：**
| 项目 | 内容 |
|------|------|
| 年龄 | 28-40 岁 |
| 背景 | HR / 管理者 / 职业转型者 |
| 阶段 | 正在上 Coach 培训课程，还没拿到认证 |
| 收入 | 中等，对价格敏感 |

**特征：**
- 刚接触教练技术，充满热情
- 在培训课上学了理论，但缺少练习机会
- 和同学互相练习，但反馈不够专业
- 想知道自己学得怎么样

**需求：**
- 大量练习机会（课后能继续练）
- 知道自己哪里做得好、哪里需要改进
- 为 ICF 认证考试做准备

**痛点：**
| 痛点 | 现状 | 我们的解决方案 |
|------|------|---------------|
| 找人练习难 | 要约同学时间，还要互相配合 | AI 模拟客户，随时可练 |
| 反馈不专业 | 同学也是新手，给不了专业反馈 | AI 基于 ICF 标准给建议 |
| 不知道自己什么水平 | 培训完了但心里没底 | ICF 认证进度追踪 |
| 真人督导太贵 | $150-300/小时 | AI Mentor 免费/低价 |

**适合的功能：** 功能 1（模拟练习场）

### 用户 C：新拿 ACC 的 Coach - 小王（核心用户）

**基本信息：**
| 项目 | 内容 |
|------|------|
| 年龄 | 30-45 岁 |
| 背景 | 刚拿到 ACC 认证，开始接客户 |
| 阶段 | 有一些客户，但还在积累经验 |
| 收入 | 兼职做 Coach，收入不稳定 |

**特征：**
- 已经有基础能力，但实战经验少
- 每次 Session 后会反思，但不确定自己分析得对不对
- 想要提升到 PCC 水平，但不知道差距在哪
- 开始有真实客户，需要记录和管理

**需求：**
- Session 结束后能快速复盘
- 专业的 Mentor 反馈（但负担不起真人督导）
- 追踪自己向 PCC 的进步
- 管理客户档案，看到来访者的成长

**痛点：**
| 痛点 | 现状 | 我们的解决方案 |
|------|------|---------------|
| 写笔记费时间 | 每次 Session 后要花 30 分钟写笔记 | 自动生成笔记摘要 |
| 不确定自己做得对不对 | 自己复盘但没有专业反馈 | 逐字稿 + 行内点评 |
| 督导太贵约不起 | 一个月才约一次督导 | 每次 Session 都有 AI Mentor |
| 看不到来访者进步 | 只有自己感觉，没有数据 | 来访者成长档案 |

**适合的功能：** 功能 1 + 功能 2（两个都需要）

### 用户 D：Coach 培训机构 - 某教练学院（B2B）

**基本信息：**
| 项目 | 内容 |
|------|------|
| 类型 | B2B 客户 |
| 规模 | 每期 20-50 名学员 |
| 痛点 | 学员练习机会不够，通过率不理想 |

**特征：**
- 课程时间有限，没法给每个学员足够练习
- 学员水平参差不齐，难以个性化指导
- 希望提高学员的 ICF 考试通过率
- 需要追踪学员的学习进度

**需求：**
- 让学员课后能自己练习
- 看到每个学员的能力数据
- 提高认证考试通过率
- 作为课程的增值服务

**痛点：**
| 痛点 | 现状 | 我们的解决方案 |
|------|------|---------------|
| 练习时间不够 | 课堂时间有限 | 学员课后用 AI 练习 |
| 无法个性化 | 一个老师对几十个学员 | AI 给每个人个性化反馈 |
| 学员水平难追踪 | 只有考试才知道 | 后台看每个学员的进度 |

**适合的功能：** 功能 1（模拟练习）+ 机构管理后台

### 用户 E：中级 Coach - 小张

**基本信息：**
| 项目 | 内容 |
|------|------|
| 年龄 | 35-50 岁 |
| 背景 | 已有 PCC 或正在冲刺 PCC |
| 阶段 | 做了 1-3 年，有稳定客户 |
| 收入 | Coach 是主要收入来源之一 |

**特征：**
- 有一定经验，但感觉遇到瓶颈
- 客户量增加，笔记管理变得繁琐
- 希望更系统地追踪每个来访者的进展
- 想要精进技术，向 MCC 发展

**需求：**
- 高效管理多个客户的档案
- 发现自己的盲区和固定模式
- 给来访者展示 TA 的成长（增加客户粘性）
- 持续精进，突破瓶颈

**痛点：**
| 痛点 | 现状 | 我们的解决方案 |
|------|------|---------------|
| 客户多了管理乱 | 用 Word/笔记本记录，找起来麻烦 | 来访者档案管理 |
| 不知道自己的模式 | 没人指出自己的盲区 | AI 分析发现固定模式 |
| 难证明自己的价值 | 来访者进步是"感觉" | 来访者成长报告可分享 |

**适合的功能：** 功能 2（实战复盘为主）

## 模块 A：Coach 目录（面向客户）

> 参考：ADPList、Psychology Today、Noomii

**Profile 包含信息：**

| 区块 | 内容 |
|------|------|
| 基本信息 | 头像、姓名、一句话介绍 |

| 价格 | 单次价格 / 套餐价格 |
| 评价 | 客户评价 + 评分 |
| 可用时间 | 显示最近可预约时段 |
| 自我介绍 | 视频/文字介绍 |

**Profile 示例：**

```
┌─────────────────────────────────────────────────────────────┐
│  [头像]   张教练                                             │
│           ICF PCC | 5年经验                                  │
│           "帮助职场人找到内心的方向"                          │
│                                                             │
│  专长：职业转型 | 领导力 | 工作生活平衡                       │
│  语言：中文、英文                                            │

│                                                             │
│  ⭐ 4.9 (28条评价)                                          │
│                                                             │
│  价格：$150/次 (60分钟)                                      │
│                                                             │
│  [ 查看可用时间 ]  [ 预约 Session ]                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### A.2 搜索与筛选

**筛选维度：**

| 筛选项 | 选项 |
|--------|------|
| 专长领域 | 职业发展 / 领导力 / 创业 / 生活平衡 / 健康 / 关系 |
| 认证等级 | ICF ACC / PCC / MCC / 其他 |
| 价格范围 | $50-100 / $100-200 / $200+ |
| 语言 | 中文 / 英文 / 其他 |
| 可用时间 | 本周 / 下周 / 指定日期 |
| 服务方式 | 线上 / 线下 / 都可以 |

### A.3 智能匹配（Phase 2）

**流程：**
```
客户填写简单问卷
    ↓
• 你想解决什么问题？
• 你偏好什么风格的 Coach？
• 你的预算范围？
    ↓
AI 推荐 3-5 个匹配的 Coach
    ↓
客户浏览 Profile，选择预约
```

### A.4 评价系统

**评价维度：**

| 维度 | 说明 |
|------|------|
| 整体评分 | 1-5 星 |
| 倾听能力 | Coach 是否真正听懂了我 |
| 提问质量 | 问题是否有启发性 |
| 推荐意愿 | 是否愿意推荐给朋友 |
| 文字评价 | 开放式反馈 |

**显示规则：**
- Session 结束 24 小时后邀请评价
- 评价需要 Coach 确认后才显示（避免恶意评价）
- 或：设置最低字数要求，自动过滤低质量评价

## 模块 B：预约系统

> 参考：ADPList、Calendly

### B.1 Coach 日历设置

**Coach 可设置：**

| 设置项 | 说明 |
|--------|------|
| 可用时段 | 每周哪些时间可以预约 |
| Session 时长 | 30分钟 / 60分钟 / 90分钟 |
| 提前预约时间 | 至少提前 24 小时 / 48 小时 |
| 每周预约上限 | 最多接受多少个预约 |
| 休息日 | 哪些日期不可预约 |

### B.2 客户预约流程

```
客户在 Coach Profile 点击"预约"
    ↓
选择 Session 类型和时长
    ↓
查看可用时间，选择时段
    ↓
填写客户信息和预约详情
    ↓
确认预约
    ↓
双方收到确认邮件/通知
```

**客户预约表单**（参考 Psychology Today）

| 字段 | 必填 | 说明 |
|------|------|------|
| 姓名 | ✅ | 客户全名 |
| 邮箱 | ✅ | 用于发送确认和提醒 |
| 电话 | ❌ | 可选，备用联系方式 |
| 想讨论的内容 | ✅ | 让 Coach 提前了解客户需求 |

**为什么要收集这些信息：**
- Coach 可以提前了解客户，做好准备
- 提高 Session 的效率和质量
- 建立客户档案，便于后续跟进

### B.3 预约管理

**Coach 端：**
- 查看所有预约（日历视图 / 列表视图）
- 确认 / 拒绝 / 改期
- 添加 Session 备注

**客户端：**
- 查看我的预约
- 取消 / 改期（需符合 Coach 政策）
- Session 前提醒

### B.4 视频会议集成

| 阶段 | 方案 |
|------|------|
| MVP | 预约后提供 Zoom/Google Meet 链接（Coach 自己创建） |
| Phase 2 | 自动生成会议链接（集成 Zoom API） |
| Phase 3 | 平台内置视频会议 |

## 模块 C：AI Supervision（面向 Coach）

> 这是平台的**差异化核心**，竞品没有的功能

### 核心理念

**不做 AI 模拟客户，做真人 Session + AI 督导反馈**

| 方式 | 说明 | 价值 |
|------|------|------|
| ~~AI 模拟客户~~ | ❌ 不做 | 练习场景是假的，学了还要面对真人 |
| **真人 Session + AI 反馈** | ✅ 做这个 | 实战中学习，每次真实对话都是成长机会 |

### 产品逻辑

```
客户通过平台预约 Coach
        ↓
进行真人 Session（Zoom/Google Meet）
        ↓
AI 实时分析对话内容
        ↓
Session 后给 Coach 督导反馈
        ↓
Coach 在实战中持续进步
```

### 功能子模块

| 功能 | 名称 | 包含 | 目标用户 |
|------|------|------|---------|
| **C.1** | 实时督导 | Session 中 AI 实时分析，给提示（可选） | 新手 Coach |
| **C.2** | 会后复盘 | 逐字稿+行内点评、ICF 评估、改进建议 | 所有 Coach |
| **C.3** | 笔记摘要 | 自动生成 Session 笔记 | 所有 Coach |
| **C.4** | 成长追踪 | 多次 Session 的能力趋势、来访者成长档案 | 长期用户 |

## 功能 C.1：实时督导（Session 中）

> 可选功能：Coach 可以选择开启或关闭

### 工作方式

```
Session 进行中
    ↓
AI 实时转录对话内容
    ↓
在 Coach 的屏幕上显示实时逐字稿 + 行内反馈（客户看不到）
    ↓
帮助 Coach 在当下做出更好的回应
```

### 实时逐字稿 + 行内反馈（核心功能）

> **设计原则：** 反馈必须和具体对话内容关联，让 Coach 知道"是针对哪句话给的建议"

**实时面板布局：**

```
┌─────────────────────────────────────────────┐
│  AI Supervision                              │
│  实时逐字稿 + 反馈（仅你可见）                 │
├─────────────────────────────────────────────┤
│                                             │
│  00:30  你                                  │
│  "今天想聊什么？"                            │
│                                             │
│  00:35  Michael                             │
│  "我最近工作压力很大..."                     │
│                                             │
│  00:50  你                                  │
│  "听起来你承受了很多。能多说说吗？"            │
│  ┌─────────────────────────────────────┐    │
│  │ ✅ 共情 + 开放式问题，做得好！         │    │
│  └─────────────────────────────────────┘    │
│                                             │
│  01:20  Michael                             │
│  "就是老板给的任务太多了..."                 │
│                                             │
│  01:35  你                                  │
│  "你有没有试过和老板沟通一下？"               │
│  ┌─────────────────────────────────────┐    │
│  │ ⚠️ 这是建议，不是提问。可以改成        │    │
│  │ "关于这个情况，你有什么想法？"          │    │
│  └─────────────────────────────────────┘    │
│                                             │
└─────────────────────────────────────────────┘
```

**为什么要显示逐字稿 + 行内反馈：**

| 原有方式 | 问题 | 改进后 |
|---------|------|--------|
| 只显示反馈列表 | 不知道是针对哪句话的反馈 | 反馈直接显示在相关对话下方 |
| 无时间戳 | 不知道什么时候给的反馈 | 每条对话都有时间戳 |
| 无说话人标识 | 分不清谁说的 | 明确标识"你"和客户名字 |

### 实时反馈类型

| 类型 | 颜色 | 图标 | 示例 |
|------|------|------|------|
| 做得好 | 绿色 | ✅ | "共情 + 开放式问题，做得好！" |
| 建议 | 黄色 | ⚠️ | "这是建议，不是提问。可以改成..." |
| 统计 | 蓝色 | 📊 | "Talk ratio: 你 35% | 客户 65%" |

### 实时统计面板

| 指标 | 说明 | 理想值 |
|------|------|--------|
| Talk Ratio | Coach vs 客户说话比例 | Coach < 40% |
| Open Questions | 开放式问题数量 | 越多越好 |
| Positive | 做得好的反馈数量 | 追踪正向行为 |
| Tips | 改进建议数量 | 了解需改进之处 |

### 隐私考虑

- 实时逐字稿和反馈只显示在 Coach 的屏幕上
- 客户不知道 AI 在分析
- Coach 需要提前告知客户"可能会录音用于提升服务质量"

## 功能 C.2：会后复盘（Session 后）

> 这是核心功能，每次 Session 后自动生成

### 工作方式

```
Session 结束
    ↓
AI 分析完整对话录音/逐字稿
    ↓
生成督导报告：
  • 逐字稿 + 行内点评
  • ICF 能力评估
  • 做得好的地方
  • 改进建议
    ↓
Coach 查看报告，获得成长
```

### 督导报告内容

| 模块 | 内容 |
|------|------|
| **逐字稿 + 行内点评** | 在具体对话位置给出反馈（做得好/可改进） |
| **ICF 能力评估** | 8 项核心能力的评分和具体解释 |
| **亮点** | 这次 Session 做得好的 2-3 点 |
| **改进建议** | 下次可以尝试的 2-3 个建议 |
| **关键时刻** | 标记对话中的重要突破点 |

### 逐字稿 + 行内点评示例

| 时间 | 说话人 | 内容 | AI 点评 |
|------|--------|------|---------|
| 00:30 | Coach | 今天想聊什么？ | |
| 00:35 | 客户 | 我最近工作压力很大... | |
| 00:50 | Coach | 听起来你承受了很多。能多说说吗？ | ✅ 共情 + 开放式问题 |
| 01:35 | Coach | 做不完的感觉是什么样的？ | ✅ 深入探索感受 |
| 02:20 | Coach | 你有没有试过和老板沟通？ | ⚠️ 这是建议，不是提问 |
| 03:25 | *沉默 8 秒* | | ✅ 给客户思考空间 |

### 1.5 适合度评估（针对考虑转型的人）

**目标用户：** 用户 A - 考虑转型做 Coach 但不确定自己是否适合的人

**功能描述：** 让用户免费体验一次 AI 模拟对话，结束后给出"你是否适合做 Coach"的评估报告。

**用户流程：**
```
注册（免费） → 选择一个简单场景 → 进行 10-15 分钟对话 → 获得适合度评估报告
```

**评估报告内容：**

| 模块 | 内容 |
|------|------|
| 整体适合度 | 高 / 中 / 需要更多练习 |
| 天赋亮点 | 你在这次对话中展现的优势（如：共情能力强、善于倾听） |
| 待发展区域 | 需要通过培训提升的能力（如：提问技巧、引导方式） |
| Coach 能力简介 | 科普 ICF 核心能力，让 TA 了解 Coach 需要什么 |
| 下一步建议 | 推荐培训课程 / 继续免费练习 |

**评估维度：**

| 维度 | 评估内容 | 说明 |
|------|---------|------|
| 倾听意愿 | 是否愿意让对方多说 | Coach 需要 70% 时间听 |
| 好奇心 | 是否追问、探索 | Coach 要保持好奇而非评判 |
| 共情表达 | 是否回应对方情绪 | Coach 要让人感到被理解 |
| 引导能力 | 是否用问题引导思考 | Coach 不给答案，而是问问题 |
| 耐心 | 是否给对方思考时间 | Coach 要学会用沉默 |

```

│  你的 Coach 适合度评估                                   │

│  ✅ 善于倾听 - 你让对方说了 68% 的时间                   │
│  ✅ 有耐心 - 你在关键时刻给了对方思考空间                 │
│                                                         │
```

### 2.1 录音转文字

**功能描述：** 记录真实的 Coach-客户对话，自动转成文字。

**功能要求：**

| 功能 | 说明 |
|------|------|
| 多语言支持 | 中文、英文、其他常用语言 |

**使用方式（从简单到理想）：**

| 方式 | 操作 | 体验 | 阶段 |
|------|------|------|------|
| **方式 1：手动上传** | Coach 自己录音，结束后上传文件 | 麻烦，但最简单实现 | MVP |
| **方式 2：AI 机器人自动加入** | 安装插件后，AI 机器人自动加入 Zoom/腾讯会议 | 省心，主流做法 | Phase 2 |
| **方式 3：原生集成** | 直接在我们的平台上开视频会议 | 最流畅，但开发成本高 | Phase 3 |

**方式 2 详细流程（推荐）：**

```
Coach 在日历上有一个 Zoom 会议
        ↓

        ↓
会议中：实时录音 + 实时转文字
        ↓
会议结束后自动生成：
  • 完整文字稿
  • Session 笔记摘要
  • Mentor 建议和分析
        ↓
Coach 打开 App 就能看到结果
```

**竞品参考：**
- [Lyssn](https://www.lyssn.io/)：HIPAA 合规录音 + 自动分析
- [tl;dv](https://tldv.io/)：AI 自动加入会议 + 转录 + 摘要
- [Read AI](https://www.read.ai/)：实时转录 + 情绪分析 + 会后建议

**客户同意机制：**
- AI 机器人加入时会显示"正在录音"提示
- Coach 需要提前告知客户并获得同意
- 可在会议开始时自动播放同意提示语

### 2.3 逐字稿 + 行内点评（核心差异化功能）

**功能描述：** 显示完整对话逐字稿，在具体位置给出点评（做得好 / 需改进），最后给出总结。

> 这是市场空白！就像老师批改作文一样，在具体的地方给反馈，而不只是最后给个分数。

#### 2.3.1 逐字稿 + 行内点评示例

| 时间 | 说话人 | 内容 | AI 点评 |
|------|--------|------|---------|
| 00:30 | Coach | 今天想聊什么？ | |
| 00:35 | 客户 | 我最近工作压力很大，不知道该怎么办... | |
| 00:50 | Coach | 听起来你承受了很多。能多说说是什么让你感到压力吗？ | ✅ **做得好**：共情 + 开放式提问 |
| 01:20 | 客户 | 就是老板给的任务太多了，我觉得自己做不完... | |
| 01:35 | Coach | 做不完的感觉是什么样的？ | ✅ **做得好**：深入探索感受 |
| 02:10 | 客户 | 就是很焦虑，晚上也睡不好... | |
| 02:20 | Coach | 你有没有试过和老板沟通一下工作量的问题？ | ⚠️ **可改进**：这是建议，不是提问。可以改成"关于这个情况，你有什么想法？" |
| 03:00 | 客户 | 我不敢... | |
| 03:05 | Coach | 为什么不敢呢？ | |
| 03:10 | 客户 | 我怕他觉得我能力不行... | |
| 03:20 | Coach | 嗯... | |
| 03:25 | *（沉默 8 秒）* | | ✅ **做得好**：给客户思考空间 |
| 03:33 | 客户 | 其实我一直都很在意别人怎么看我... | |
| 03:45 | Coach | 这个"在意别人怎么看"，是从什么时候开始的？ | ✅ **做得好**：有力提问，引导客户探索更深层次 |

#### 2.3.2 总结报告

**Session 概览：**

| 项目 | 内容 |
|------|------|
| 时长 | 32 分钟 |
| 主题 | 工作压力 → 发现核心是"在意他人评价" |
| 客户突破 | 意识到压力背后是对他人看法的担忧 |

**做得好的地方：**
- ✅ 开场建立了安全感，客户愿意敞开
- ✅ 03:25 的沉默运用很好，让客户自己说出了关键洞察
- ✅ 03:45 的提问引导客户探索了更深层的原因

**可以改进的地方：**
- ⚠️ 02:20 给了建议而不是提问（Coach 不是顾问）
- ⚠️ 建议更多探索"在意别人看法"这个主题

**能力评分：**

| 能力 | 评分 | 说明 |
|------|------|------|
| 积极倾听 | 85% | 多次准确反映客户情绪 |
| 有力提问 | 70% | 大部分是好问题，有一次给了建议 |
| 培养信任 | 90% | 客户明显感到安全，愿意分享 |
| 唤起觉察 | 80% | 帮助客户发现了"在意他人评价"的模式 |

| 5 | 保持临在状态 | Maintains Presence | 全身心专注当下，跟随客户 |
| 6 | 积极倾听 | Listens Actively | 真正听到客户说的和没说的 |
| 7 | 唤起觉察 | Evokes Awareness | 帮助客户产生新的洞察 |
| 8 | 促进客户成长 | Facilitates Client Growth | 推动客户采取行动，实现改变 |

#### 为什么用 ICF 标准？

| 原因 | 说明 |
|------|------|
| 权威性 | ICF 是全球最大的教练认证机构，71,000+ 认证 Coach |
| 标准化 | 有明确的行为指标，可以客观评估 |
| 被认可 | Coach 培训和认证都基于这个框架 |
| 可对比 | 用户可以对照 ICF 标准了解自己的水平 |

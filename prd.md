## 市场价值验证

### 这个生意有市场吗？

**结论：有，而且市场正在快速增长。**

#### 教练行业市场数据（2025）

| 指标 | 数据 | 来源 |
|------|------|------|
| **全球教练市场规模** | $53.4 亿美元 | ICF 2025 全球研究 |
| **北美市场** | $28.9 亿美元 | ICF |
| **全球Coach数量** | 122,974 人（+15%） | ICF |
| **年增长率** | 8.5% CAGR | 行业报告 |
| **预计2032年规模** | $95 亿美元 | 行业预测 |

#### 为什么这是个好时机？

| 趋势 | 说明 | 对我们的意义 |
|------|------|-------------|
| **Coach数量激增** | 2019-2023增长54% | 更多人需要培训工具 |
| **认证需求增加** | 85%客户要求ICF认证 | Coach必须练习才能通过认证 |
| **培训成本高** | 培训课程$3,000-$12,000 | 需要低成本练习补充 |
| **督导费用高** | $150-300/小时 | AI替代方案有需求 |
| **AI教练工具兴起** | 预计2026年超$10亿 | 市场接受度在提升 |

#### 类似产品验证（SimCare AI）

SimCare AI 在**心理咨询培训**领域已验证这个模式：

| 验证点 | SimCare数据 | 说明 |
|--------|------------|------|
| **用户接受度** | 3周2,500用户 | 需求真实存在 |
| **机构买单** | 30+高校试点 | B2B模式可行 |
| **收入增长** | ~$4M ARR | 商业模式跑通 |
| **投资认可** | YC + $200万种子轮 | 资本看好 |

#### 我们的差异化机会

| 维度 | 现状 | 我们的机会 |
|------|------|-----------|
| **目标市场** | SimCare做心理咨询 | Coach市场是蓝海 |
| **评估标准** | 没有针对ICF标准的AI工具 | ICF是唯一权威标准 |
| **中国市场** | 几乎没有竞品 | 可先占领中文市场 |
| **实战复盘** | 现有工具只做模拟练习 | 我们做模拟+实战分析 |

## 一、产品概述

### 1.1 产品名称
**AI Coach Mentor**（暂定）

### 1.2 一句话描述
帮助教练（Coach）提升专业技能的 AI 训练与反馈工具。

### 1.3 核心价值
| 用户痛点 | 我们的解决方案 |
|---------|---------------|
| 想转型但不知道自己适不适合做 Coach | 免费体验 + 适合度评估 |
| 找真人练习太贵（$150-300/小时） | AI 模拟客户，随时免费练习 |
| 不知道自己哪里需要改进 | 基于 ICF 标准自动分析，给出建议 |
| 写 Session 笔记太费时间 | 自动生成笔记摘要 |
| 约督导时间难协调 | 24/7 随时可用 |
| 在真人面前暴露不足有心理压力 | AI 无评判，私密安全 |
| 看不到自己帮助来访者的成效 | 来访者成长档案，可视化 TA 的进步 |

### 1.4 目标用户概览

| 优先级 | 用户群 | 适合的功能 | 付费意愿 |
|--------|--------|-----------|---------|
| ⭐⭐⭐⭐ | 考虑转型的人 | 功能 1（免费体验） | 免费 → 转化 |
| ⭐⭐⭐⭐⭐ | Coach 培训学员 | 功能 1（模拟练习） | 中 |
| ⭐⭐⭐⭐⭐ | 新拿 ACC 的 Coach | 功能 1 + 功能 2 | 中高 |
| ⭐⭐⭐⭐ | Coach 培训机构 (B2B) | 功能 1 + 管理后台 | 高（批量采购） |
| ⭐⭐⭐ | 中级 Coach（1-3年） | 功能 2（实战复盘） | 高 |

## 二、目标用户画像（Personas）

### 用户 A：考虑转型做 Coach 的人 - 小陈

**基本信息：**
| 项目 | 内容 |
|------|------|
| 年龄 | 30-45 岁 |
| 背景 | HR / 培训师 / 管理者 / 心理咨询相关 |
| 阶段 | 听说过教练，感兴趣，但不确定自己适不适合 |
| 收入 | 稳定工作，考虑职业转型或发展副业 |

**特征：**
- 对"帮助他人成长"有热情
- 听朋友或同事提过教练，觉得自己可能适合
- 但不确定自己是否具备做 Coach 的特质
- 培训课程要几万块，不敢轻易报名

**核心问题：**
> "我适合做 Coach 吗？我有没有这方面的天赋？"

**需求：**
- 低成本/免费试一试教练对话是什么感觉
- 了解 Coach 需要什么能力
- 得到一个初步评估：我适不适合

**痛点：**
| 痛点 | 现状 | 我们的解决方案 |
|------|------|---------------|
| 不知道自己适不适合 | 只能听别人说，没法亲自体验 | 免费试用 AI 模拟对话 |
| 培训太贵不敢轻易报名 | 几万块的培训费，万一不适合呢 | 先体验，再决定 |
| 不了解 Coach 需要什么能力 | 网上信息零散 | 体验后给出"适合度评估" |

**适合的功能：** 功能 1（免费体验版）

**商业价值：** 这是获客漏斗的入口
```
免费体验 → 发现自己适合 → 去报培训班 → 成为付费用户
```

**基本信息：**

**特征：**
- 刚接触教练技术，充满热情

**需求：**
- 大量练习机会（课后能继续练）

**痛点：**
| 痛点 | 现状 | 我们的解决方案 |
|------|------|---------------|
| 找人练习难 | 要约同学时间，还要互相配合 | AI 模拟客户，随时可练 |
| 反馈不专业 | 同学也是新手，给不了专业反馈 | AI 基于 ICF 标准给建议 |
| 不知道自己什么水平 | 培训完了但心里没底 | ICF 认证进度追踪 |
| 真人督导太贵 | $150-300/小时 | AI Mentor 免费/低价 |

**适合的功能：** 功能 1（模拟练习场）

## 功能 1：模拟练习场

### 1.1 AI 模拟客户（被教练者）

**功能描述：** AI 扮演不同类型的"客户"，让 Coach 练习教练对话。

**客户人设库：**

| 人设类型 | 示例场景 |
|---------|---------|
| 职业发展 | "我想跳槽但又害怕失败" |
| 领导力挑战 | "我的团队不听我的话" |
| 工作生活平衡 | "我太忙了，没时间陪家人" |
| 人际关系 | "我和同事相处不好" |
| 自信心 | "我总觉得自己不够好" |
| 转型期 | "我想换行业但不知道做什么" |
| 拖延症 | "我总是拖到最后一刻" |
| 压力管理 | "我快要崩溃了" |

**客户性格变量：**
- 开放程度：健谈 ↔ 沉默寡言
- 情绪状态：平静 ↔ 焦虑/激动
- 配合程度：积极 ↔ 抵触
- 自我觉察：高 ↔ 低

---

### 1.2 实时反馈（练习中）

**功能描述：** 在练习对话过程中，给 Coach 实时提示。

> 参考产品：Poised（沟通实时反馈，$13-19/月）

**反馈维度：**

| 维度 | 说明 | 示例提示 |
|------|------|---------|
| 提问质量 | 是否使用开放式问题 | "试试用'什么'或'如何'开头的问题" |
| 倾听比例 | Coach vs 客户说话时间 | "你说话占比 70%，试着多听" |
| 沉默运用 | 是否给客户思考空间 | "可以等一等，让对方多想想" |
| 共情表达 | 是否反映客户情绪 | "客户刚才表达了焦虑，可以回应一下" |
| 打断次数 | 是否打断客户 | "注意：刚才打断了对方" |

**显示方式：**
- 私密显示（只有 Coach 自己能看到）
- 不干扰对话流程
- 可选择开启/关闭

**参考来源：** Poised 的实时会议反馈功能

---

### 1.3 练习报告（练习后）

**功能描述：** 每次练习结束后，生成详细分析报告。

**报告内容：**

```
┌─────────────────────────────────────────────┐
│           练习报告 - 2025.01.06              │
├─────────────────────────────────────────────┤
│                                             │
│  总体评分: 75/100                            │
│                                             │
│  ┌─────────────────────────────────────┐    │
│  │ ICF 核心能力评估                     │    │
│  ├─────────────────────────────────────┤    │
│  │ 积极倾听      ████████░░  80%       │    │
│  │ 有力提问      ██████░░░░  60%       │    │
│  │ 创造觉察      ███████░░░  70%       │    │
│  │ 促进成长      █████░░░░░  50%       │    │
│  └─────────────────────────────────────┘    │
│                                             │
│  亮点:                                       │
│  • 第3分钟的提问很有力量                      │
│  • 共情表达做得很好                          │
│                                             │
│  改进建议:                                   │
│  • 多给客户思考的时间                         │
│  • 避免连续追问，一次问一个问题                │
│                                             │
│  关键时刻回放:                               │
│  • 02:30 - 很好的开放式问题                   │
│  • 05:45 - 这里可以深入探索                   │
│  • 08:20 - 客户有突破，抓住了！               │
│                                             │
└─────────────────────────────────────────────┘
```

#### 1.3.1 评分解释（核心功能）

> **设计原则：** 每个评分都必须有具体解释，让 Coach 知道"为什么是这个分数"，而不是只看到一个数字。

**评分解释示例：**

```
┌─────────────────────────────────────────────────────────┐
│  积极倾听  ████████░░  80%                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  📊 为什么是 80%？                                       │
│                                                         │
│  ✅ 做得好（+分）:                                       │
│     • 你让客户说话占比 65%（理想值 >60%）                  │
│     • 3 次准确复述了客户的关键词                          │
│     • 在 04:20 捕捉到客户的情绪变化并回应                  │
│                                                         │
│  ⚠️ 可以改进（-分）:                                     │
│     • 在 02:15 和 08:30 打断了客户（共 2 次）              │
│     • 客户说"我觉得..."时，你没有追问感受                  │
│                                                         │
│  💡 下次试试:                                            │
│     当客户说到一半停顿时，多等 3-5 秒再说话                 │
│                                                         │
│  📖 什么是"积极倾听"？                                   │
│     ICF 定义：全身心关注客户所说和未说的，                 │
│     理解其背景含义，支持客户自我表达。                     │
│     → 点击查看详细说明和练习技巧                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**每个 ICF 能力的评分都包含：**

| 模块 | 内容 |
|------|------|
| 分数 | 百分比 + 进度条可视化 |
| 为什么是这个分 | 具体列出加分点和扣分点 |
| 证据 | 引用对话中的具体时间点和原话 |
| 改进建议 | 给出可操作的下一步 |
| 能力解释 | 解释这个 ICF 能力的定义，帮助 Coach 理解标准 |

### 1.4 进度追踪与成长可视化（长期）

**功能描述：** 追踪 Coach 的长期进步，可视化成长曲线，增加成就感。

> **设计目标：** 让 Coach 看到自己的成长，保持学习动力。

#### 1.4.1 能力成长曲线

**周/月/季度对比视图：**

```
┌─────────────────────────────────────────────────────────┐
│           我的成长轨迹                                   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  📈 积极倾听 - 过去 3 个月                               │
│                                                         │
│  100% ┤                                                 │
│   90% ┤                              ╭───●  本月 88%    │
│   80% ┤                    ╭────────╯                   │
│   70% ┤          ╭────────╯                             │
│   60% ┤    ●────╯                                       │
│   50% ┤   首次 55%                                      │
│       └────┬────────┬────────┬────────┬────────→        │
│           1月      2月       3月      4月               │
│                                                         │
│  🎉 进步了 33%！                                         │
│                                                         │
│  进步最快的时期: 2月第2周（练习了5次后突破70%）            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 1.4.2 能力雷达图（五边形战士图）

**什么是雷达图：** 就像游戏里看角色属性的那种五边形/多边形图，每个角代表一项能力，越往外代表能力越强。

**我们的雷达图展示 5 项核心能力：**

| 能力 | 说明 | 首次 | 现在 | 变化 |
|------|------|------|------|------|
| 积极倾听 | 真正听到对方说的和没说的 | 55% | 80% | ↑25% |
| 有力提问 | 问出让人深思的问题 | 50% | 72% | ↑22% |
| 培养信任 | 创造安全的对话氛围 | 60% | 85% | ↑25% |
| 保持临在 | 全身心专注当下 | 45% | 70% | ↑25% |
| 唤起觉察 | 帮助对方产生新洞察 | 40% | 68% | ↑28% |

**雷达图的价值：**
- 一眼看出哪些能力是强项（图形往外凸）
- 一眼看出哪些能力需要提升（图形往内凹）
- 对比"首次"和"现在"两条线，看到整体成长

#### 1.4.3 ICF 认证进度追踪（替代游戏化徽章）

> **为什么不做游戏化徽章？**
> - 竞品 SimCare AI 没有做徽章系统
> - "连续7天练习"这种徽章对专业 Coach 没有实际价值
> - Coach 真正关心的是：**我离 ICF 认证还有多远？**

**ICF 官方认证要求：**

| 级别 | 教育时长 | 教练经验 | 能力水平 |
|------|---------|---------|---------|
| ACC（初级） | 60+ 小时 | 100+ 小时 | 聚焦客户"问题" |
| PCC（专业） | 125+ 小时 | 500+ 小时 | 聚焦"问题"+"人" |
| MCC（大师） | 200+ 小时 | 2,500+ 小时 | 聚焦"整个人" |

**我们可以追踪的进度：**

| 追踪项 | 说明 | 对 Coach 的价值 |
|--------|------|----------------|
| 练习时长累计 | 记录在平台上的练习时长 | 可作为 ICF 认证"教练经验"的补充证明 |
| 能力水平对标 | 对比 ICF 的 PCC Markers 评估 | 知道自己是 ACC/PCC/MCC 哪个水平 |
| 弱项专项突破 | 识别最需要提升的能力 | 针对性准备 ICF 考试 |
| 模拟评估报告 | 按 ICF 标准生成评估 | 类似 ICF 认证考试的录音评审 |

**能力水平对标示例：**

| 能力 | 你的水平 | ACC 标准 | PCC 标准 | 差距 |
|------|---------|---------|---------|------|
| 积极倾听 | 72% | 60% | 80% | 距 PCC 还差 8% |
| 有力提问 | 65% | 60% | 80% | 距 PCC 还差 15% |

**这样设计的好处：**
- 让 Coach 知道"我现在是什么水平"
- 明确"我要达到 PCC 还需要提升什么"
- 练习时长可以作为申请 ICF 认证的辅助材料

### 1.5 适合度评估（针对考虑转型的人）

**目标用户：** 用户 A - 考虑转型做 Coach 但不确定自己是否适合的人

**功能描述：** 让用户免费体验一次 AI 模拟对话，结束后给出"你是否适合做 Coach"的评估报告。

**用户流程：**
```
注册（免费） → 选择一个简单场景 → 进行 10-15 分钟对话 → 获得适合度评估报告
```

**评估报告内容：**

| 模块 | 内容 |
|------|------|
| 整体适合度 | 高 / 中 / 需要更多练习 |
| 天赋亮点 | 你在这次对话中展现的优势（如：共情能力强、善于倾听） |
| 待发展区域 | 需要通过培训提升的能力（如：提问技巧、引导方式） |
| Coach 能力简介 | 科普 ICF 核心能力，让 TA 了解 Coach 需要什么 |
| 下一步建议 | 推荐培训课程 / 继续免费练习 |

**评估维度：**

| 维度 | 评估内容 | 说明 |
|------|---------|------|
| 倾听意愿 | 是否愿意让对方多说 | Coach 需要 70% 时间听 |
| 好奇心 | 是否追问、探索 | Coach 要保持好奇而非评判 |
| 共情表达 | 是否回应对方情绪 | Coach 要让人感到被理解 |
| 引导能力 | 是否用问题引导思考 | Coach 不给答案，而是问问题 |
| 耐心 | 是否给对方思考时间 | Coach 要学会用沉默 |

### 2.1 录音转文字

**功能描述：** 记录真实的 Coach-客户对话，自动转成文字。

**功能要求：**

| 功能 | 说明 |
|------|------|
| 多语言支持 | 中文、英文、其他常用语言 |
| 说话人识别 | 区分 Coach 和客户的发言 |
| 高准确率 | 专业术语识别准确 |
| 隐私保护 | 端到端加密，数据安全 |

**使用方式（从简单到理想）：**

| 方式 | 操作 | 体验 | 阶段 |
|------|------|------|------|
| **方式 1：手动上传** | Coach 自己录音，结束后上传文件 | 麻烦，但最简单实现 | MVP |
| **方式 2：AI 机器人自动加入** | 安装插件后，AI 机器人自动加入 Zoom/腾讯会议 | 省心，主流做法 | Phase 2 |
| **方式 3：原生集成** | 直接在我们的平台上开视频会议 | 最流畅，但开发成本高 | Phase 3 |

**方式 2 详细流程（推荐）：**

```
Coach 在日历上有一个 Zoom 会议
        ↓
AI 机器人自动加入会议（像 tl;dv、Read AI 那样）
        ↓
会议中：实时录音 + 实时转文字
        ↓
会议结束后自动生成：
  • 完整文字稿
  • Session 笔记摘要
  • Mentor 建议和分析
        ↓
Coach 打开 App 就能看到结果
```

**竞品参考：**
- [Lyssn](https://www.lyssn.io/)：HIPAA 合规录音 + 自动分析
- [tl;dv](https://tldv.io/)：AI 自动加入会议 + 转录 + 摘要
- [Read AI](https://www.read.ai/)：实时转录 + 情绪分析 + 会后建议

**客户同意机制：**
- AI 机器人加入时会显示"正在录音"提示
- Coach 需要提前告知客户并获得同意
- 可在会议开始时自动播放同意提示语

### 附录 A：ICF 核心能力模型

> **来源说明：** ICF（International Coaching Federation，国际教练联盟）是全球最权威的教练认证机构。
> 以下 8 项核心能力是 ICF 官方定义的标准，所有 ICF 认证考试都基于这个框架评估。
>
> **官方链接：** https://coachingfederation.org/core-competencies

#### ICF 8 项核心能力（2021年更新版）

| # | 能力名称 | 英文 | 简单解释 |
|---|---------|------|---------|
| 1 | 展现道德实践 | Demonstrates Ethical Practice | 遵守职业道德和保密原则 |
| 2 | 体现教练心态 | Embodies a Coaching Mindset | 保持好奇、开放、不评判的态度 |
| 3 | 建立和维护协议 | Establishes and Maintains Agreements | 明确目标，让双方知道要做什么 |
| 4 | 培养信任和安全感 | Cultivates Trust and Safety | 创造安全的对话空间 |
| 5 | 保持临在状态 | Maintains Presence | 全身心专注当下，跟随客户 |
| 6 | 积极倾听 | Listens Actively | 真正听到客户说的和没说的 |
| 7 | 唤起觉察 | Evokes Awareness | 帮助客户产生新的洞察 |
| 8 | 促进客户成长 | Facilitates Client Growth | 推动客户采取行动，实现改变 |

#### 为什么用 ICF 标准？

| 原因 | 说明 |
|------|------|
| 权威性 | ICF 是全球最大的教练认证机构，71,000+ 认证 Coach |
| 标准化 | 有明确的行为指标，可以客观评估 |
| 被认可 | Coach 培训和认证都基于这个框架 |
| 可对比 | 用户可以对照 ICF 标准了解自己的水平 |

#### SimCare AI 详细信息（参考产品）

> **为什么参考 SimCare：** SimCare 在心理咨询培训领域验证了"AI模拟练习"的市场需求。我们要做的是**教练（Coach）领域的SimCare**。

| 项目 | 数据 |
|------|------|
| **成立时间** | 2024年（YC S24 批次） |
| **创始背景** | 芝加哥大学辍学生，因付不起$9,000/小时的临床沟通培训费而创业 |
| **早期验证** | 上线 3 周：2,500 用户，$5,000 月收入 |
| **机构客户** | 30+ 试点（宾大、弗吉尼亚理工、悉尼大学等） |
| **用户群体** | 心理咨询学校、远程治疗公司、社工项目、医学院 |
| **估计年收入** | ~$4M ARR（2025年数据） |
| **虚拟来访者** | 500+（专业咨询师设计角色档案） |
| **反馈维度** | 语气、共情、沟通、非语言暗示、诊断准确性、临床技能 |
| **定价** | $79/年 |
| **融资** | 2025年2月完成 $200 万种子轮（YC + Drive Capital 领投） |
| **创建方式** | 咨询师设计角色档案 → AI 即兴扮演（非传统数据训练）|

**SimCare 与我们的差异：**

| 维度 | SimCare | AI Coach Mentor |

| 评估标准 | 临床诊断准确性 | ICF 核心能力 |
